## 如何拍攝截圖

### 方法一：瀏覽器開發者工具
1. 按 F12 開啟開發者工具
2. 按 Ctrl+Shift+P (Windows) 或 Cmd+Shift+P (Mac)
3. 輸入 "screenshot" 並選擇 "Capture full size screenshot"

### 方法二：瀏覽器擴充套件
推薦使用 "FireShot" 或 "Full Page Screen Capture" 等擴充套件

### 方法三：系統截圖工具
- Windows: Win + Shift + S
- Mac: Cmd + Shift + 4
- Linux: Shift + PrtScr
